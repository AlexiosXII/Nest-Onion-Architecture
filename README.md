## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository designed in [Onion Architecture](<(https://medium.com/expedia-group-tech/onion-architecture-deed8a554423)>).

## Installation

```bash
$ bun install
```

## Running the app

```bash
# development
$ bun run start

# watch mode
$ bun run start:dev

# production mode
$ bun run start:prod
```

## Test

```bash
# unit tests
$ bun run test

# e2e tests
$ bun run test:e2e

# test coverage
$ bun run test:cov
```
